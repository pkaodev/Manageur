# Manageur

## Installation


```bash
#  Main
curl -H "Accept: application/vnd.github.v3.raw" -L "https://api.github.com/repos/pkaodev/Manageur/contents/scripts/setup_with_curl_zip.bash" | bash

# Branch
curl -H "Accept: application/vnd.github.v3.raw" -L "https://api.github.com/repos/pkaodev/Manageur/contents/scripts/setup_with_curl_zip.bash?ref=<__BRANCH__>" | bash
```
